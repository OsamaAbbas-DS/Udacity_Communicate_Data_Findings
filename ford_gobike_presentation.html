Ford GoBike Data Analysis¶

by Osama Abbas¶

Investigation Overview¶

What do you think would affect a user's trip duration with the Ford GoBike bike sharing service? Is it the age? Their gender? Could it be the user type (one-time customer or service subscriber)? Or maybe the time of the day when they started the trip? We will try to answer all of these questions and more throughout our analysis of the Ford GoBike dataset.

Dataset Overview¶

The dataset is for all trips taken during February of 2019, with 16 features recorded, including: the duration, start and end time, start and end station plus their coordinates, user birth year, user gender, etc.. and 183412 records.

def mean_poly(x, y, bins = 5, **kwargs):
    # set bin edges if none or int specified
    if type(bins) == int:
        bins = np.linspace(x.min(), x.max(), bins+1)
    bin_centers = (bin_edges[1:] + bin_edges[:-1]) / 2

    # compute counts
    data_bins = pd.cut(x, bins, right = False,
                       include_lowest = True)
    means = y.groupby(data_bins).mean()

    # plot the data
    plt.errorbar(x = bin_centers, y = means, **kwargs)

Conclusion¶

In conclusion, we now have a better understanding of each feature's effect on the duration of the bike share service trip. We came up with eye-openning insights about how different segments of users benefit from our service. For instance, we have discovered that gender does not really affect the duration of the trip, but variables such as: user type, age, and the start hour of the trip do. We also have a better understanding of users' purpose when they use our service, as most use it for workday commute. However, users who use our service for other purposes seem to have relatively higher trip durations. All of this would help us target the right customer, depending on the company objectives.
